# QATS
Quality Assurance Testing Support for exploratory tests

Simple Automation made in Python with the purpose of assisting QAs who are on the process of exploratory tests.
This project was made as a Undergraduate Thesis for Computer Engineering course on FAMETRO University.

This automation includes:
• EventLogger to collect both mouse and keyboard commands;
• Screen Recorder fully implemented for proof of evidence;
• Basic GUI with implemented timer synced with all functionalities.
